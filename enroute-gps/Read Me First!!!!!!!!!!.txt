Setting up Environment for running our python gprs map script:
##############################################################

You need the following software and modules installed:
*****************************************
1.Python27 installer for windows
2.pynmea(Python NMEA Library - a small library to help read and parse NMEA data)
3.python-dateutil(Provides powerful extensions to the standard datetime module)
4.pyserial-2.7.win32(Multiplatform Serial Port Module for Python)
5.numpy-1.6.1-win32-superpack-python3.2(Scientific tools for Python)
6.matplotlib-1.3.1.win32-py2.7(A python plotting library, making publication quality plots )
7.six-1.4.1(Python 2 and 3 compatibility utilities)
8.pyparsing-2.0.1.win32-py2.7

Most of this module can be downloaded at http://www.lfd.uci.edu/~gohlke/pythonlibs/
To Install  any of the module follow this link:http://docs.python.org/2/using/windows.html

Note:If you will need to run the python scripts under dos command prompt you will have to set the path variables after installing python
####
*Setting Path Variable for running python scripts:http://docs.python.org/2/using/windows.html
